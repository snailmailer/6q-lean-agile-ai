# 6Q Lean Agile AI

**6Q Lean Agile AI** is an intelligent decision-support assistant that combines the structure of **Lean Six Sigma's DMAIC methodology** with the adaptability of **Agile** development. 

This application guides users through six focused questions to clarify problems, analyze solutions, and execute improvements effectively.

## Features

- **The 6-Step Wizard**: A guided flow covering Define, Measure, Analyze, Improve, and Control phases.
- **Contextual AI Tips**: Every step provides specific advice tailored to Lean/Agile principles.
- **Auto-Save**: Progress is saved automatically to your local browser storage—never lose your thought process.
- **Premium UI**: A modern, dark-mode interface with glassmorphism effects and smooth transitions.
- **Summary Dashboard**: Review your entire decision framework in a single view and print/export it.

## Getting Started

This is a **Vanilla HTML/CSS/JS** application. No build tools or servers are required.

### How to Run

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.

That's it!

## Project Structure

- `index.html`: Main entry point and layout.
- `css/style.css`: All styles, variables, and animations.
- `js/app.js`: Core application logic, wizard state management, and data configuration.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Variable-based theming, Flexbox/Grid layouts.
- **JavaScript (ES6+)**: Vanilla logic for maximum performance and portability.
- **Lucide Icons**: Beautiful, open-source icons.
